# Compliance-as-Code: A Technical Guide for GRC Professionals

A comprehensive, searchable technical reference covering 40+ topics across cloud architecture, FedRAMP 20x, OSCAL, security scanning, DevSecOps, and AI for compliance.

**By Rupinder Pal Singh** | CISSP, CISM, CISA, CRISC

## What's Inside

- **Compliance Tools** — Steampipe, Prowler, OPA, Vanta, Aikido, and how they work together
- **FedRAMP 20x & OSCAL** — KSI model, timeline, OSCAL deep dive, tools landscape, FIPS endpoints
- **Developer Tools** — GitHub/GitLab, CI/CD, Terraform, Docker, Kubernetes
- **Security Scanning** — SAST, DAST, SCA, secrets detection, SBOMs
- **Cloud Architecture (AWS)** — VPC, peering, PrivateLink, multi-account, security services, Bedrock
- **Security Concepts** — IAM/RBAC/Zero Trust, SIEM, KMS, Entra ID, API keys
- **Architecture Patterns** — Microservices, API Gateway, Facade pattern, serverless, DevSecOps/GitOps
- **AI & ML for Compliance** — RAG, vector databases, MCP, building enterprise RAG systems
- **FAQ** — 16 real questions from GRC professionals learning technical concepts
